2.2 Hover Effect Recommended ‚≠ê‚≠ê‚≠ê‚≠ê‚òÜ

Move up on hover to create floating effect

Try Me
CSS Code 
/* Hover effect - element moves up and adds shadow on hover */

/* Base styles */
.card {
    /* 1. position: relative
       - Although only top is changed here, setting relative is good practice
       - Ensures element occupies space in normal document flow
       - Convenient for future absolute positioned children */
    position: relative;

    /* 2. transition: all 0.3s
       - Defines transition animation, smooth changes within 0.3s
       - Makes top and box-shadow changes animated */
    transition: all 0.3s;

    /* 3. cursor: pointer
       - Shows pointer cursor on hover
       - Indicates interactive element */
    cursor: pointer;
}

/* Hover state */
.card:hover {
    /* Same position: relative maintains positioning context */
    position: relative;

    /* top: -5px
       - Moves up 5px relative to original position
       - Negative value moves up, positive moves down */
    top: -5px;

    /* box-shadow: 0 10px 20px rgba(0,0,0,0.2)
       - Adds shadow effect
       - 0: horizontal offset | 10px: vertical offset | 20px: blur radius
       - rgba(0,0,0,0.2): semi-transparent black */
    box-shadow: 0 10px 20px rgba(0,0,0,0.2);
}

2.3 As Reference Element Recommended ‚≠ê‚≠ê‚≠ê‚≠ê‚òÜ

relative is the best choice for absolute positioning reference

Parent position: relative
Child
position: absolute
üí° Why use relative as reference?
Absolute elements look up for the first non-static ancestor as positioning reference.
Not required to use relative, but relative doesn't leave document flow, making it the safest choice.
CSS Code 
/* ‚úÖ Best practice: parent sets relative as reference */
.parent {
    position: relative;  /* üëà Doesn't leave document flow, safest choice */
    width: 300px;
    height: 150px;
}

/* Child: absolute positioning */
.child {
    position: absolute;
    top: 10px;
    right: 10px;
    /* Positions relative to parent ‚úÖ */
}

/* ‚ö†Ô∏è Risk: parent doesn't set positioning */
.parent {
    /* No position set */
}

.child {
    position: absolute;
    top: 10px;
    /* Positions relative to browser window ‚ö†Ô∏è */
}

CSS Code

    position: absolute;
    top: 50%;
    left: 50%;

}
