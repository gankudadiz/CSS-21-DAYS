3.2 Attribute Exact Match Popular ‚≠ê‚≠ê‚≠ê‚≠ê‚òÜ

3.3 Attribute Start/End Match Common ‚≠ê‚≠ê‚≠ê‚òÜ‚òÜ

Scenario: External link icons, file type icons, https indicators

Baidu (has href, left border) Google (new window, light blue bg) GitHub (https start, green) PDF file (yellow bg)
CSS Code 
/* Attribute value starts with https */

/* Attribute value ends with .pdf */
a[href$=".pdf"] { background-color: #ffeaa7; }

/* Attribute value contains specified value */
a[target="_blank"] { background-color: #e8f4fd; }

4. Pseudo-classes

4.1 User Action Pseudo-classes Essential ‚≠ê‚≠ê‚≠ê‚≠ê‚≠ê

Scenario: Button hover animations, form validation feedback, link click feedback



CSS Code 
/* Hover state */
.btn:hover {
    background-color: #2980b9;
    transform: translateY(-2px);
    box-shadow: 0 4px 8px rgba(0,0,0,0.2);

/* Active state */

4.2 Structural Pseudo-classes Popular ‚≠ê‚≠ê‚≠ê‚≠ê‚òÜ

Scenario: Table zebra stripes, nav active states, list first/last styles

  • First (:first-child, blue)
  • Second
  • Third (:nth-child(3), red)
  • Fourth (even, light gray blue border)
  • Fifth (odd, white red border)
  • Last (:last-child, green)
CSS Code 
/* First child */
li:first-child {
    background-color: #3498db;
    color: white;
}

/* Last child */
li:last-child {
    background-color: #27ae60;
    color: white;
}

/* Third child */
li:nth-child(3) {
    background-color: #e74c3c;
    color: white;
}

/* Even children */
li:nth-child(even) {
    background-color: #f8f9fa;
    border-left: 4px solid #3498db;
}

6. Selector Specificity

Specificity Calculation Table

Selector Type Weight
!importantHighest (use with caution)
Inline style (style attribute)1000
ID selector (#id)100
Class, Attribute, Pseudo-class10
Element, Pseudo-element1

Practical Demo Essential ‚≠ê‚≠ê‚≠ê‚≠ê‚≠ê

Scenario: Resolve style conflicts, debug CSS

Class selector .text (blue)

Element + class p.text (green, higher specificity)

ID + class #special-text (purple, highest)

CSS Code 
/* Specificity: 10 */
.priority-result .text { color: blue; }

/* Specificity: 11 (1+10) */
.priority-result p.text { color: green; }

/* Specificity: 110 (100+10) */
#special-text.text { color: purple; }

/*
 * Final result: purple (highest specificity)
 * Specificity calculation: ID(100) + class(10) = 110
 */
