Core Concept: Flex Container & Flex Item Relationship

Flex Container

Is the parent element, set display: flex to enable

→ contains ←
Flex Item

Is the child element, placed directly inside the container

Key Point: The container controls the overall layout of all items; items are controlled by the container.

1. Flex Container Properties

Properties on the flex container that control how child elements (Flex Items) are arranged

1.1 display Popular ⭐⭐⭐⭐⭐

display: flex or inline-flex to enable Flexbox layout

flex block container
inline-flex inline
CSS Code 
/* Block-level flex container */
.container {
    display: flex;
}

/* Inline flex container */
.container {
    display: inline-flex;
}

/*
 * Difference:
 * - flex: Container as block-level (独占一行)
 * - inline-flex: Container as inline (与其他行内元素同行)
 */

1.2 flex-direction Popular ⭐⭐⭐⭐⭐

Set main axis direction (element arrangement direction)

row (default): left→right

1 2 3

column: top→bottom

1 2 3
CSS Code 
/* Main axis direction */
.container {
    flex-direction: row;             /* left→right (default) */
    flex-direction: row-reverse;     /* right→left */
    flex-direction: column;          /* top→bottom */
    flex-direction: column-reverse;  /* bottom→top */
}

/* row-reverse effect:
 * ┌───┐┌───┐┌───┐
 * │ 3 ││ 2 ││ 1 │  ← Elements reversed
 * └───┘└───┘└───┘
 */

1.3 justify-content Popular ⭐⭐⭐⭐⭐

Main axis alignment (horizontal/vertical depends on flex-direction)

flex-start: near start

1 2 3

center: center

1 2 3

space-between: distribute

1 2 3

space-around: equal space on both sides

1 2 3

space-evenly: equal space everywhere

1 2 3
CSS Code 
/* Main axis alignment */
.container {
    justify-content: flex-start;     /* Near start (default) */
    justify-content: center;         /* Center */
    justify-content: flex-end;       /* Near end */
    justify-content: space-between;  /* Distribute, equal space between items */
    justify-content: space-around;   /* Equal space on both sides of items */
    justify-content: space-evenly;   /* Equal space everywhere */
}

/*
 * space-around vs space-evenly:
 * space-around:  Edge space is half of item space
 * space-evenly:  All spaces equal
 */

1.4 align-items Popular ⭐⭐⭐⭐⭐

Cross axis (vertical) alignment

flex-start: align to top

Top Top Top

center: vertical center

Center Center Center

flex-end: align to bottom

Bottom Bottom Bottom

stretch (default): stretch to fill container height

Auto Stretch Fill
CSS Code 
/* Cross axis alignment */
.container {
    align-items: stretch;      /* Stretch to fill (default) */
    align-items: flex-start;   /* Near cross axis start */
    align-items: center;       /* Center */
    align-items: flex-end;     /* Near cross axis end */
    align-items: baseline;     /* Item baseline alignment */
}

/* Note: stretch works when items have no height set */

1.5 flex-wrap Popular ⭐⭐⭐⭐☆

Whether items wrap (items 80px wide, container 280px)

nowrap (default): no wrap, items shrink

1 2 3 4 5 6

wrap: auto wrap, keep original width

1 2 3 4 5 6 7 8

wrap-reverse: wrap, first row at bottom

1 2 3 4 5 6 7 8
CSS Code 
/* Wrap control */
.container {
    flex-wrap: nowrap;        /* No wrap, items shrink (default) */
    flex-wrap: wrap;          /* Wrap */
    flex-wrap: wrap-reverse;  /* Wrap, first row at bottom */
}

/* wrap-reverse effect:
 * ┌───┐┌───┐
 * │ 5 ││ 6 │  ← First row at bottom
 * ├───┼───┤
 * │ 1 ││ 2 │
 * │ 3 ││ 4 │
 * └───┴───┘
 */

1.6 gap Popular ⭐⭐⭐⭐⭐

Space between items (recommended, simpler than margin)

gap: 20px

Equal Spacing Clean
CSS Code 
/* Item spacing */
.container {
    gap: 10px;              /* All directions */
    gap: 10px 20px;         /* row-gap column-gap */
    row-gap: 10px;          /* Row gap only */
    column-gap: 20px;       /* Column gap only */
}

/* Recommended to use gap instead of margin:
 * - No need to handle first/last item margin
 * - Simpler code
 * - Easier to maintain
 */

Core Concept: Flex Item & Flex Container Relationship

Flex Item

Is the child element, placed directly inside the container

← belongs to ←
Flex Container

Is the parent element, the "home" of all Flex Items

Key Point: Flex Item properties (like flex-grow, align-self) only affect the current element, used to fine-tune individual items

2. Flex Item Properties

Properties on flex items to control individual item behavior (relative to the container's overall layout)

2.1 flex-grow Popular ⭐⭐⭐⭐☆

Item grow factor (occupy remaining space)

flex: 1 - equal distribution

1 2 3

ratio 1:2:1

1 2 3
CSS Code 
/* Grow factor */
.item {
    flex-grow: 0;     /* Default: no grow */
    flex-grow: 1;     /* Occupy remaining space */
    flex-grow: 2;     /* Twice the grow ratio of 1 */
}

/* Equal distribution */
.container > * {
    flex: 1;
}

/* 1:2:1 ratio */
.item:nth-child(1) { flex-grow: 1; }
.item:nth-child(2) { flex-grow: 2; }
.item:nth-child(3) { flex-grow: 1; }

/*
 * flex-grow only distributes remaining space
 * If items have fixed width, set flex-basis: 0
 */

2.2 flex-shrink Common ⭐⭐⭐☆☆

Item shrink factor

flex-shrink: 0 - no shrink (fixed width)

No Shrink No Shrink No Shrink

⚠️ Container width 320px, three 120px items need 360px total, but they won't shrink

CSS Code 
/* Shrink factor */
.item {
    flex-shrink: 1;   /* Default: can shrink */
    flex-shrink: 0;   /* Cannot shrink */
    flex-shrink: 2;   /* Larger shrink ratio */
}

/* Fixed width, no shrink */
.sidebar {
    flex: 0 0 200px;  /* No grow, no shrink, fixed 200px */
}

2.3 order Common ⭐⭐⭐☆☆

Change item order (doesn't affect DOM structure)

A B C

Display order: B → C → A

CSS Code 
/* Order */
.item {
    order: 0;     /* Default value */
    order: -1;    /* Comes before */
    order: 1;     /* Comes after */
}

/* Example: adjust navigation order */
.nav-item:nth-child(1) { order: 3; } /* Home last */
.nav-item:nth-child(2) { order: 1; }
.nav-item:nth-child(3) { order: 2; }

/*
 * order doesn't change HTML order in DOM
 * Only changes visual order
 * Search engines and screen readers still read HTML order
 */

2.4 align-self Popular ⭐⭐⭐⭐☆

Individual item cross axis alignment (overrides container's align-items)

Top Center Bottom
CSS Code 
/* Individual item alignment */
.item {
    align-self: auto;        /* Inherit container's align-items */
    align-self: flex-start;  /* Near cross axis start */
    align-self: center;      /* Center */
    align-self: flex-end;    /* Near cross axis end */
    align-self: stretch;     /* Stretch */
    align-self: baseline;    /* Baseline alignment */
}

/* Use case: specific item needs special alignment */
.nav-item:last-child {
    align-self: flex-end;  /* Last item to right/bottom */
}

2.5 flex shorthand Popular ⭐⭐⭐⭐⭐

Set grow, shrink, and basis at once

flex: 1 - equal distribution

Equal Width Equal Width Equal Width

flex: 0 0 fixed width

Fixed Adaptive Fixed
CSS Code 
/* flex shorthand: grow shrink basis */
.item {
    flex: 0 1 auto;       /* Default: no grow, can shrink, based on size */
    flex: 1;              /* flex: 1 1 0% - equal distribution */
    flex: auto;           /* flex: 1 1 auto - auto size + equal */
    flex: none;           /* flex: 0 0 auto - no flexing */
}

/* Recommended写法 */
.sidebar {
    flex: 0 0 200px;  /* No grow, no shrink, fixed 200px */
}

.content {
    flex: 1;          /* Occupy remaining space */
}

/*
 * flex-basis has priority over width
 * flex: 1 equals flex: 1 1 0%
 */

3. Common Layouts

Flexbox common layout patterns in actual development

3.1 Center Horizontally & Vertically Popular ⭐⭐⭐⭐⭐

Most commonly used centering layout

Centered Content
CSS Code 
/* Center horizontally and vertically - most common */
.container {
    display: flex;
    justify-content: center;  /* Horizontal center */
    align-items: center;      /* Vertical center */
    min-height: 100vh;        /* Need height */
}

/*
 * Principle:
 * justify-content controls main axis (horizontal) alignment
 * align-items controls cross axis (vertical) alignment
 * Combined for perfect centering
 */

3.2 Navigation Bar Layout Popular ⭐⭐⭐⭐⭐

Logo + Menu items + Button

Logo
Home Products About
Login
CSS Code 
/* Navigation bar layout */
.navbar {
    display: flex;
    justify-content: space-between;  /* Split to ends */
    align-items: center;             /* Vertical center */
    padding: 10px 20px;
    background: #3498db;
}

.nav-links {
    display: flex;
    gap: 20px;                       /* Menu item spacing */
    list-style: none;
    margin: 0;
    padding: 0;
}

.nav-links a {
    color: white;
    text-decoration: none;
}

3.3 Three Column Layout Popular ⭐⭐⭐⭐⭐

Sidebar + Main content area

Left Main Content
(Adaptive) Right
CSS Code 
/* Three column layout */
.layout {
    display: flex;
}

.sidebar-left {
    flex: 0 0 200px;  /* Fixed width, no flex */
}

.content {
    flex: 1;          /* Occupy all remaining space */
}

.sidebar-right {
    flex: 0 0 150px;  /* Fixed width, no flex */
}

/* Responsive: hide sidebars on mobile */
@media (max-width: 768px) {
    .sidebar-left, .sidebar-right {
        display: none;
    }
}

3.4 Responsive Card Grid Popular ⭐⭐⭐⭐⭐

Auto wrap + Adaptive width

Card 1 Card 2 Card 3 Card 4
CSS Code 
/* Responsive card grid */
.card-grid {
    display: flex;
    flex-wrap: wrap;     /* Auto wrap */
    gap: 20px;           /* Card spacing */
}

.card {
    flex: 1 1 280px;     /* Min 280px, auto expand */
    max-width: 400px;    /* Max width limit */
}

/*
 * flex: 1 1 280px meaning:
 * - grow: 1     → Occupy remaining space
 * - shrink: 1   → Shrink when space insufficient
 * - basis: 280px → Base width 280px
 */
