Explore advanced usage of order and align-self properties

The order property controls the visual order of items on the main axis. Smaller values come first, negative values are allowed, and the original HTML structure remains unchanged.

The align-self property sets the cross-axis alignment for a single item, overriding the container’s align-items. Typical values include flex-start, center, flex-end, and stretch.

Item 3

.item:nth-child(2) { order: 0; align-self: auto; }

Core Concept: order Property

The order property controls the visual order of Flex Items in the container. It does not change the DOM structure HTML order, only the visual arrangement.

Important: Search engines and screen readers still follow the original HTML order. Don't rely on order for accessibility ordering.

Order Property Details

Smaller order values appear first; larger values appear later

2.1 Order Basic Usage Popular ⭐⭐⭐⭐☆

Adjust item display order

HTML Order: A → B → C, display order changes after setting order

A
order:3 B
order:1 C
order:2

Display Order: B → C → A

CSS Code 
/* Order basic usage */
.item:nth-child(1) { order: 3; }  /* 3rd position */
.item:nth-child(2) { order: 1; }  /* 1st position */
.item:nth-child(3) { order: 2; }  /* 2nd position */

/* HTML order is still A → B → C
 * But visual order becomes B → C → A
 */

/* Order default value */
.item {
    order: 0;  /* Default, all items default to 0 */
}

/* Negative values come first */
.item:first-child { order: -1; }  /* Comes first */

Core Concept: align-self Property

The align-self property controls the alignment of a single item on the cross axis, and can override the container's align-items setting.

Use Case: When most items use the same alignment but one item needs special alignment, use align-self.

align-self Property Details

3.1 align-self Basic Usage Popular ⭐⭐⭐⭐☆

Single item cross axis alignment

Container align-items: flex-start, middle item uses align-self: center

Top Center Bottom
CSS Code 
/* align-self basic usage */
.container {
    display: flex;
    align-items: flex-start;  /* All items top aligned by default */
    height: 200px;
}

.item:nth-child(2) {
    align-self: center;       /* Middle item centered */
}

.item:nth-child(3) {
    align-self: flex-end;     /* Bottom item bottom aligned */
}

/* All align-self values */
.item {
    align-self: auto;        /* Inherit align-items */
    align-self: flex-start;  /* Cross axis start */
    align-self: center;      /* Cross axis center */
    align-self: flex-end;    /* Cross axis end */
    align-self: stretch;     /* Stretch (when no height) */
    align-self: baseline;    /* Baseline alignment */
}

CSS Code 
/* Payment methods container:
 * 1. Use flex to place cards in a row
 * 2. gap controls spacing between cards
 * 3. flex-wrap: wrap allows cards to wrap onto new lines
 */
.payment-methods {
    display: flex;     /* Enable Flexbox layout */
    gap: 15px;         /* Horizontal and vertical spacing between cards */
    flex-wrap: wrap;   /* Wrap to next line when width is not enough */
}

/* Single payment card:
 * flex: 1 1 120px is equivalent to:
 *   flex-grow: 1   → card can grow to fill remaining space
 *   flex-shrink: 1 → card can shrink to avoid overflow
 *   flex-basis: 120px → base width of 120px
 */
.payment-card {
    flex: 1 1 120px;        /* Minimum width 120px, then auto expand */
    background: #f8f9fa;    /* Light background color for contrast */
    border: 2px solid #e0e0e0;  /* Default border color */
    border-radius: 8px;     /* Rounded corners for card */
    padding: 15px;          /* Inner spacing for better readability */
    text-align: center;     /* Center text inside the card */
    cursor: pointer;        /* Pointer cursor to indicate clickable area */
    transition: all 0.3s;   /* Smooth hover transition */
}

/* Hover state: highlight border to give visual feedback */
.payment-card:hover {
    border-color: #3498db;
}

/* Selected state:
 * 1. Green border to highlight current selection
 * 2. Soft green background to indicate active option
 */
.payment-card.selected {
    border-color: #27ae60;
    background: #f0fff4;
}

/* Responsive product list container:
 * 1. Use flex + flex-wrap to simulate a grid
 * 2. gap controls spacing between product cards

.product-list {
    display: flex;      /* Horizontal Flex layout */
    flex-wrap: wrap;    /* Wrap items onto new lines */
    gap: 15px;          /* Spacing between rows and columns */
}

/* Single product card:
 * flex: 1 1 200px
 *   → base width is about 200px
 *   → remaining space is shared between items
 * max-width: 300px prevents cards from becoming too wide
 */
.product-item {
    flex: 1 1 200px;        /* Base width 200px, flexible growth and shrink */
    max-width: 300px;       /* Limit maximum width to 300px */
    background: white;      /* White background for content */
    border: 1px solid #e0e0e0;  /* Subtle border around each card */
    border-radius: 8px;     /* Rounded corners */
    overflow: hidden;       /* Hide overflow to keep rounded corners clean */
}

/* Single column layout on small screens:
 * 1. flex-basis set to 100% so each card occupies full width
 * 2. max-width: 100% ensures it never exceeds container width
 */
@media (max-width: 480px) {
    .product-item {
        flex: 1 1 100%;     /* Each card takes one full row */
        max-width: 100%;    /* Match container width */
    }
}

Practice 3: Responsive Form Field Groups

Use flex-wrap for adaptive forms

CSS Code 
/* Form field group container:
 * 1. Use flex-wrap to allow fields to wrap on small screens
 * 2. gap controls spacing between fields
 */
.form-group {
    display: flex;        /* Place fields in a row */
    flex-wrap: wrap;      /* Wrap to next line when needed */
    gap: 15px;            /* Space between fields */
    margin-bottom: 15px;  /* Vertical spacing between groups */
}

/* Single form field container:
 * flex: 1 1 200px
 *   → evenly distributes space between fields
 *   → minimum width is about 200px
 * flex-direction: column puts label above input
 */
.form-field {
    flex: 1 1 200px;        /* Base width and flexible behavior */
    display: flex;
    flex-direction: column; /* Label above, input below */
}

/* Wider fields:
 * flex-grow is 2 so it takes twice the width of normal fields
 * suitable for fields like "Detailed Address"
 */
.form-field.wide {
    flex: 2 1 400px;        /* Field that spans two columns visually */
}

/* Label styles:
 * 1. Smaller font size to save space
 * 2. Gray color to reduce visual weight
 */
.form-field label {
    font-size: 12px;
    color: #666;
    margin-bottom: 5px;     /* Space between label and input */
}

/* Input styles:
 * 1. Padding improves readability and click target
 * 2. Border + radius match overall card style

CSS Code 
/* Step indicator outer container:
 * 1. Use flex so each step is aligned horizontally
 * 2. space-between spreads steps evenly across the row
 */
.step-indicator {
    display: flex;
    justify-content: space-between;  /* Distribute steps across full width */
    margin-bottom: 30px;             /* Spacing below the indicator */
}

/* Single step item:
 * 1. Place step number and label vertically
 * 2. flex: 1 gives each step equal width
 * 3. position: relative for the connector line (::after)
 */
.step-item {
    display: flex;
    flex-direction: column;
    align-items: center;
    flex: 1;               /* All steps share width equally */
    position: relative;
}

/* Connector line between steps:
 * 1. Draw a horizontal line using a pseudo-element
 * 2. Positioned in the vertical center of the circle
 */
.step-item::after {
    content: '';
    position: absolute;
    top: 20px;         /* Align with the center of the circle */
    left: 50%;         /* Start from center of current step */
    width: 100%;       /* Extend towards the next step */
    height: 2px;
    background: #e0e0e0;  /* Default gray connector line */
}

/* No connector line on the last step */
.step-item:last-child::after {
    display: none;
}

/* Completed steps show green connector line */
.step-item.completed::after {
    background: #27ae60;
}

/* Circular step number:
 * 1. Fixed width and height to form a circle
 * 2. Use flex to center the number inside
 */
.step-number {
    width: 40px;
    height: 40px;
    border-radius: 50%;      /* Make it a circle */
    background: #e0e0e0;
    display: flex;
    align-items: center;
    justify-content: center;
    font-weight: bold;
    color: white;
    z-index: 1;              /* Ensure it sits above the connector line */
}

Flex Items Advanced Learning Summary

Key Points: order Property

  • order: 0 is default, smaller values come first
  • Supports negative values, negative values come before positive
  • Does not affect DOM structure and accessibility
  • Suitable for order adjustment in responsive layouts

Key Points: align-self Property

  • Overrides container's align-items setting
  • Values: auto | flex-start | center | flex-end | stretch | baseline
  • stretch requires item without fixed height
  • Often used for special alignment of single items

Common Application Scenarios

  • Mobile priority for important buttons (order)
  • Navbar ordering adjustment (order)
  • Notification icon alignment (align-self: flex-start)
  • Form button centering (align-self: center)
  • Responsive card grid (flex: 1 1)
