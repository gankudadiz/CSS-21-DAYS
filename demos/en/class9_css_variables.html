Define & Use Variables Core ‚≠ê‚≠ê‚≠ê‚≠ê‚≠ê

CSS variables use `--` prefix and are called with var().

CSS Variable Demo

This is a card demo using CSS variables.

CSS Code 
/* Define variables */
:root {
    --primary-color: #3498db;
    --secondary-color: #27ae60;
    --spacing-md: 16px;
    --border-radius: 8px;
}

/* Use variables */
.card {
    background-color: var(--primary-color);
    padding: var(--spacing-md);
    border-radius: var(--border-radius);
}

/* Fallback value */
.button {
    background-color: var(--btn-bg, #3498db);
}

Variable Scope Important ‚≠ê‚≠ê‚≠ê‚≠ê‚òÜ

CSS variables are inherited and can be overridden at different selector levels.

Parent Scope
Child Override - Uses red
Inherits Parent - Uses blue
CSS Code 
/* Global scope */
:root {
    --primary-color: #3498db;
}

/* Local scope */
.card {
    --card-bg: #ffffff;
    background-color: var(--card-bg);
}

/* Override parent */
.special-card {
    --card-bg: #f8f9fa;
}

Theme Switching Essential ‚≠ê‚≠ê‚≠ê‚≠ê‚≠ê

Click buttons to switch themes. CSS variables respond in real-time.

Theme Switching Demo

This card changes when you switch themes.

JS Code 
function switchTheme(themeName) {
    const demoBox = document.getElementById('theme-demo-box');

    switch(themeName) {
        case 'light':
            demoBox.style.setProperty('--theme-bg', '#ffffff');
            demoBox.style.setProperty('--theme-text', '#333333');
            demoBox.style.setProperty('--theme-border', '#dddddd');
            break;
        case 'dark':
            demoBox.style.setProperty('--theme-bg', '#1a1a1a');
            demoBox.style.setProperty('--theme-text', '#f0f0f0');
            demoBox.style.setProperty('--theme-border', '#444444');
            break;
        case 'custom':
            demoBox.style.setProperty('--theme-bg', 'linear-gradient(135deg, #667eea, #764ba2)');
            demoBox.style.setProperty('--theme-text', '#ffffff');
            break;
    }
}

Responsive Variables Advanced ‚≠ê‚≠ê‚≠ê‚òÜ‚òÜ

Responsive Variables Demo

Resize the browser to see spacing and font changes.

Current: Desktop

CSS Code 
/* Base variables */
.responsive-demo {
    --responsive-padding: 16px;
    --responsive-font: 16px;
    padding: var(--responsive-padding);
    font-size: var(--responsive-font);
}

/* Mobile override */
@media (max-width: 600px) {
    .responsive-demo {
        --responsive-padding: 10px;
        --responsive-font: 14px;
    }
}

JavaScript Control Useful ‚≠ê‚≠ê‚≠ê‚≠ê‚òÜ

Read and modify CSS variables in real-time with JavaScript.

JS Control Demo

Adjust the card style with sliders:

JS Code 
function updateVar(varName, value, displayId) {
    const demoBox = document.getElementById('js-demo-box');
    demoBox.style.setProperty('--' + varName, value);

    if (displayId) {
        document.getElementById(displayId).textContent = value;
    }
}
